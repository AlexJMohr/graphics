# Alex Mohr Graphics

3D OpenGL graphics project. See the Features section below.

## Dependencies

- Java 10 (lower versions might work)
- LWJGL 3.2.1

## Features

- [x] Model Loading
- [x] Textures
- [ ] Basic Lighting (ambient, diffuse, specular)
- [ ] Shadows
- [ ] Skeletal Animation
- [ ] Particles
- [ ] 2D Font Rendering

## Build and run
```bash
./gradlew build
./gradlew run
```
