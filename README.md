# Alex Mohr Graphics

3D OpenGL graphics project. See the Features section below.

## Dependencies

- Java 10 (lower versions might work)
- LWJGL 3.2.1

## Features

- [x] Model Loading
- [x] Textures
- [x] Basic Lighting (single light source)
- [ ] Advanced Lighting
    - [ ] Directional Light
    - [ ] Point Light
    - [ ] Spot Light
    - [ ] Multiple Lights
- [ ] Shadows
- [ ] Skeletal Animation
- [ ] Particles
- [ ] 2D Font Rendering
- [ ] Sky Box

## Build and run
```bash
./gradlew build
./gradlew run
```
